# C Learning Projects
This repository contains small projects that I built while learning the C programming language.
The purpose of this repository is to practice programming logic, problem solving, and basic C syntax.

## Projects
### Grade Analyzer
A simple program that calculates the average grade of students.
Features:
- Input number of students
- Input grades
- Calculate the average grade
- Display pass or fail result

## How to Compile
Use GCC compiler:
gcc grade-analyzer/main.c -o grade-analyzer

## How to Run
./grade-analyzer
